# EdgeEye Dataset Workspace

This directory is the local workspace for dataset preparation.

Large raw, staging, processed, download, and cache directories are ignored by git.
Commit only lightweight metadata, source notes, conversion scripts, and manifests.

## Target Classes

Phase 1 main classes:

- `insulator_normal`
- `insulator_surface_damage`
- `transformer_normal`
- `transformer_surface_damage`

Phase 2 environment anomaly candidate:

- `fire`

## Directory Layout

```text
dataset/
  raw/        # downloaded archives and extracted original datasets
  staging/    # manually inspected and remapped intermediate data
  processed/  # final YOLO-ready train/val/test split
  docs/       # source notes and download manifests
```

## Current Local Inventory

### Processed EdgeEye Detector V1 Dataset

- Local path: `dataset/processed/edgeeye-detector-v1/`
- Generated by: `training/prepare_dataset.py --overwrite`.
- Format: Ultralytics YOLO train/val/test split.
- Target classes:
  - `insulator_normal`
  - `insulator_surface_damage`
  - `transformer_normal`
  - `transformer_surface_damage`
- Current full conversion result:
  - Train split: 8,491 images, 8,491 label files, 112,112 boxes.
  - Validation split: 1,030 images, 1,030 label files, 14,985 boxes.
  - Test split: 890 images, 890 label files, 18,285 boxes.
  - Total: 10,411 images, 10,411 label files, 145,382 boxes.
- Detailed cleanup report: `dataset/docs/edgeeye-detector-v1-report.md`.

### Aerial Power Infrastructure Detection Train Dataset

- Local path: `dataset/raw/aerial-power-infrastructure-detection-train/`
- Source: <https://github.com/supervisely-ecosystem/aerial-power-infrastructure-detection-train-dataset>
- Size: about 266 MB.
- Split: 1,296 train images and 144 validation images.
- Annotation format: Supervisely JSON.
- Source labels: `insulator`, `broken`, `pollution-flashover`.
- EdgeEye use: seed data for `insulator_normal` and `insulator_surface_damage`.

### Insulator-Defect Detection DatasetNinja Archive

- Local archive: `dataset/insulator-defect-detection-DatasetNinja.tar`
- Extracted path: `dataset/raw/insulator-defect-detection/`
- Size: about 2.5 GB.
- Annotation format: Supervisely-style JSON.
- Split: 1,296 train images, 144 validation images, 160 test images.
- Source labels used: `insulator`, `broken`, `pollution-flashover`.
- EdgeEye use: additional `insulator_normal` and `insulator_surface_damage` data.

### Transformer Station Detection YOLOv8 Archive

- Local archive: `dataset/Transformer Station Detection.v1i.yolov8.zip`
- Extracted path: `dataset/raw/transformer-station-detection/`
- Size: about 30 MB.
- Annotation format: YOLOv8.
- Split: 353 train images, 57 validation images, 29 test images in the source archive.
- Source labels: `damage`, `graffiti`, `normal`.
- EdgeEye use: `normal` -> `transformer_normal`, `damage` -> `transformer_surface_damage`.
- Excluded from detector-v1: `graffiti`.

### Substation Equipment 15-class YOLO Archive

- Local archive: `dataset/24060960.zip`
- Extracted path: `dataset/raw/substation-equipment-15class/`
- Size: about 8.9 GB.
- Contents: nested archives `misc.zip`, `agv_night_light.zip`, `agv_night_dark.zip`, `agv_day.zip`, plus `classes.txt`.
- Extracted nested directories: `misc/`, `agv_night_light/`, `agv_night_dark/`, `agv_day/`.
- Current extracted inventory: 7,531 images, 7,537 YOLO label files, 15 source classes.
- Confirmed detector-v1 mapping:
  - `Glass disc insulator` -> `insulator_normal`.
  - `Porcelain pin insulator` -> `insulator_normal`.
  - `Power transformer` -> `transformer_normal`.
- Current use: included in `edgeeye-detector-v1` after remapping only the confirmed source classes into the current 4-class detector contract.
