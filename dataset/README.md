# EdgeEye Dataset Workspace

This directory is the local workspace for dataset preparation.

Large raw, staging, processed, download, and cache directories are ignored by
git. Commit only lightweight metadata, source notes, conversion scripts, and
manifests.

## Target Classes

Phase 1 main classes:

- `insulator_normal`
- `insulator_surface_damage`
- `transformer_normal`
- `transformer_surface_damage`

Phase 2 environment anomaly candidate:

- `fire`

## Directory Layout And Cleanup Rules

```text
dataset/
  downloads/    # source archives read by prepare_dataset.py
  raw/          # extracted original datasets; do not hand-edit
  staging/      # temporary/manual inspection workspace
  processed/    # generated YOLO-ready train/val/test datasets
  cache/        # reproducible local cache
  docs/         # tracked source notes and dataset reports
```

Cleanup policy:

- Keep source archives under `dataset/downloads/`. The converter also supports
  the previous top-level archive paths for compatibility, but new archive work
  should use:
  - `dataset/downloads/24060960.zip`
  - `dataset/downloads/Transformer Station Detection.v1i.yolov8.zip`
  - `dataset/downloads/insulator-defect-detection-DatasetNinja.tar`
- Treat `dataset/raw/` as immutable source extraction output. Re-extract from
  archives instead of hand-editing raw files.
- Treat `dataset/processed/` as generated output. Regenerate with scripts
  instead of manual edits.
- Keep `dataset/staging/`, `dataset/downloads/`, and `dataset/cache/` ignored.
  Their role is documented here so future dataset work has a stable place to
  land.
- Do not delete raw datasets, source archives, or processed datasets as part of
  routine cleanup without an explicit backup/removal decision.

- Split and sampling policy:
  - Duplicate grouping happens before train/val/test assignment.

    normal-only images.
  - Train-only damage repetition added 620 repeated damage-positive images,
    with at most one additional copy per damage-positive image.
  - Validation and test are not resampled.
