# EdgeEye Web

React + Vite frontend for EdgeEye member 5 work: dashboard, realtime inspection, fault center, report center, and later end-to-end demo flows.

## Run

```bash
bun install
bun run dev
```

Default frontend URL:

```text
http://localhost:5173
```

The API client defaults to:

```text
http://localhost:8000/api
```

Override with `VITE_API_BASE_URL` when needed.

## Build

```bash
bun run build
```
